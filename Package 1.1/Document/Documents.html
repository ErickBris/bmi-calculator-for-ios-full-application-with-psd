BMI Calculator iOS Application

First of all, thank you for purchasing our product. It will be our pleasure to fullfil your requirement related to this product. This document for those who are looking to make changes in iOS BMI Calculator application such as icon, data, sharing link, sharing data, more applications.

1. ICON SET UP

Icon is must to upload application on apple store. Bellow is the list of icon name with it's size. It must be in PNG file format.

  • Store Icon.png: 1024 x 1024 pixel (for AppStore)
  • Icon.png: 57 x 57 pixel
  • Icon@2x.png: 114 x 114 pixel
  • Icon-72.png: 72 x 72 pixel
  • Icon-72@2x.png: 144 x 144 pixel
  • Icon-76.png: 76 x 76 pixel
  • Icon-120.png: 120 x 120 pixel
  • Icon-152.png: 152 x 152 pixel
  • Icon-83.5@2x.png: 167x 167 pixel

ICON folder is placed inside the Supporting Files in XCode. See the bellow image.

2. Application Title Name

This is the text which will be shown at the top of the screen. In this project, there is a Static.h file. Open Static.h header file in XCode. Here you will be see #define APPLICATION_TITLE_NAME @. Update this text after @ and between double quotes " ". Currently text is like "BMI CALCULATOR". See the bellow image.

3. Social App Sharing Text

This is the text which will be shown while sharing app with users. In this project, there is a Static.h file. Open Static.h header file in XCode. Here you will be see #define APPLICATION_SHARE_TEXT @. Update your social sharing text after @ and between double quotes " ". Currently text is like "APPLICATION SHARING TEXT". See the bellow image.

4. Social App Sharing Link

This is the link which will be show while sharing app with users.

Open Static.h header file in XCode. Here you will be see #define SHARE_APP_LINK @. Update your social sharing link after @ and between double quotes " ". Currently link is like "https://itunes.apple.com/us/app". See the bellow image.

5. More App

This is the link which will use to show developer more application on app store.

Open Static.h header file in XCode. Here you will be see #define MORE_APP_LINK @. Update your developer link after @ and between double quotes " ". Currently link is like "itmss://itunes.apple.com/us/developer/developer-name". See the bellow image.

6. Setup Rate The Application

The application include to rate the application(Appirator) and to do it, it's required Apple ID which you will find on itunesconnect after setup of your profile and application title on apple store.

Open Static.h header file in XCode. Here you will be see #define APPID @. Update Apple ID after @ and between double quotes " ". Currently ID is like "XXXXXXXXXXX". See the bellow image.

7. Setup About BMI Text

There is a HTML file called 'About BMI.html' in Supporting Files directory, Setup text as per requirements here.

8. Admob ID Setup

This application include Admob ads integration. If you want to show ads in your application, you require AdMob ID for Banner and Interstitial(Full size Ads) which you will get from AdMob. You need to set up account if you are new user. Add ID of AdMob in static.h file.
